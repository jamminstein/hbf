# hbf

> Daft Punk acid bass with roulette grid

## Controls

- **E1** — BPM
- **E2** — root note (C2–C4)
- **E3** — filter cutoff
- **K2** — play / stop toggle
- **K3** — randomize synth patch

## Grid

8×16 roulette: [1,1] = PLAY, [1,2] = STOP, remaining cells shuffled with scales, chords, patterns, effects, BPM, gate, velocity, swing, and filter parameters.

## Requirements

- norns
- grid (16x8 recommended)
- engine: PolyPerc (built-in)

## Install

```
;install https://github.com/jamminstein/hbf
```
